# Customer Churn Analysis

## Project Overview

This project analyzes customer churn behavior using the Telco Customer Churn dataset. The goal is to identify key factors that influence customer churn and provide insights to help businesses improve customer retention.

## Tools Used

* SQL (Data exploration and analysis)
* Excel (Data cleaning)
* Power BI (Data visualization and dashboard creation)

## Dataset

Telco Customer Churn Dataset containing information about customer demographics, services used, billing details, and churn status.

Total Customers: **7043**

## Key Business Questions

* What is the overall customer churn rate?
* Which contract types have the highest churn?
* Which payment methods are associated with higher churn?
* Does internet service type influence churn?
* How do monthly charges and tenure affect churn?

## Key Insights

* Overall churn rate is **26.5%**, meaning about **1 in 4 customers leave the service**.
* Customers with **month-to-month contracts have the highest churn rate**.
* **Electronic check payment method shows the highest churn** among customers.
* Customers using **fiber optic internet service show higher churn compared to DSL users**.
* Customers who churn stay for **~18 months on average**, while retained customers stay **~38 months**.

## Dashboard

The Power BI dashboard visualizes key churn drivers including contract type, payment method, internet service type, tenure, and monthly charges.

## Project Files

* SQL queries used for analysis
* Power BI dashboard file
* Dataset used for analysis

## Business Recommendation

* Encourage customers to shift to **long-term contracts** to reduce churn.
* Investigate issues with **electronic check payments** that may cause customer dissatisfaction.
* Improve onboarding and engagement for **new customers**, as churn is highest during early tenure.
